## Detailed Project Description (For Project Section)

**HBTU College Study Platform**
*Full-Stack Educational Management System | React Native • Next.js • Supabase • PostgreSQL*

- Architected and developed a scalable educational platform serving 13 engineering branches (CSE, IT, ECE, EE, ME, CE, etc.) with support for 4 academic years and 8 semesters, enabling centralized content management and student engagement

- Built cross-platform mobile application using React Native and Expo SDK 54, implementing tab-based navigation, real-time data synchronization, Google Drive file integration, and offline-first architecture with pull-to-refresh functionality

- Developed responsive admin dashboard using Next.js 15 with TypeScript, Tailwind CSS, and Radix UI components, featuring comprehensive analytics, content management system (CMS), and role-based access control

- Implemented secure authentication system using Supabase Auth with email OTP (passwordless), JWT session management, and protected routes for both mobile and web applications

- Designed and optimized PostgreSQL database schema with 15+ relational tables, implementing Row Level Security (RLS) policies for data isolation, user permissions, and admin-only content management

- Created comprehensive content management features including notes upload/verification, PYQ (Previous Year Questions) management, syllabus document distribution, event creation with poster uploads, and opportunity posting (jobs/internships)

- Built real-time analytics dashboard with data visualization using Recharts, tracking user engagement, download metrics, content performance, branch-wise statistics, and weekly activity trends

- Implemented file storage system using Supabase Storage with CDN-backed delivery, supporting multiple file types (PDF, DOC, PPT, images) with download tracking and Google Drive integration

- Developed personalized academic tools including CGPA calculator with semester-wise grade tracking, weekly timetable viewer with day-wise scheduling, exam countdown timers with urgency indicators, and event RSVP functionality

- Created common learning resources module featuring categorized content (DSA Notes, Development Resources, Placement Preparation, AI Tools) with topic-based organization and difficulty levels

- Implemented notification system with push notifications support, in-app notification center, unread message tracking, and admin notification templates

- Built user management system with profile management, branch/semester selection, admin role assignment, user analytics, and activity tracking

- Developed search and filtering capabilities across notes, events, and opportunities with subject-wise, branch-wise, and semester-wise filtering

- Created download tracking system with per-click event logging, user analytics, file size tracking, and download history

- Implemented responsive design patterns ensuring optimal user experience across iOS, Android, and web platforms with consistent UI/UX using Lucide icons and Material Design principles

- Optimized database queries with proper indexing, implemented efficient data fetching with React Query, and reduced API response times through query optimization

- Deployed mobile application using Expo EAS (Expo Application Services) with support for development, preview, and production builds for both iOS and Android platforms

- Configured CI/CD pipeline for admin dashboard with Vercel integration, enabling automatic deployments on code push
